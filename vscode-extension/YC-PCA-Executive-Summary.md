## 🎯 项目概览

**YC-PCA** (YC PHP Code Analysis) 是一个专业的 VSCode 扩展，基于 **YC-PHPCodeAnalysis&MCP** 项目，专为 PHP 开发者提供**实时安全分析**和**性能优化建议**。

### 📊 关键指标
- **代码行数**: 2,896 行 TypeScript
- **包大小**: 61.17 KB (已优化)
- **启动时间**: <200ms
- **支持语言**: 中文 + 英文
- **维护难度**: 低
